# JobAggregator / Vacancy Parser
## Описание проекта
__JobAggregator__ — это консольное приложение на Java, предназначенное для автоматического сбора (парсинга) вакансий с внешних сайтов по расписанию. Система разработана для упрощения поиска работы: она самостоятельно обходит заданные ресурсы, фильтрует актуальные предложения и сохраняет их в базу данных для последующего анализа или уведомления пользователя.

На данный момент реализован парсер для раздела Java Developer на популярной платформе career.habr.com.
## Как это работает
* 1.Планировщик задач
Приложение запускается автоматически с заданной периодичностью (например, раз в минуту). Интервал запуска гибко настраивается в файле конфигурации app.properties.

* 2.Сбор данных (Парсинг)
При каждом запуске программа обращается к сайту career.habr.com и анализирует страницы с вакансиями:

  * URL для анализа: https://career.habr.com/vacancies/java_developer

  * Глубина поиска: первые 5 страниц выдачи.

  * Фильтрация: система отбирает только те вакансии, которые явно относятся к направлению Java (анализируя заголовок и требования).

  * Сохранение результатов
  Все найденные и отфильтрованные вакансии записываются в базу данных. Механизм записи исключает дублирование данных: если вакансия уже есть в базе (например, висит с прошлого запуска), она не будет добавлена повторно.
## Технологический стек
  * Язык: Java (версия 8+)

* Сборка: Maven

* База данных: PostgreSQL (или любая другая SQL БД, подключаемая через JDBC)

* Логирование: Slf4j + Log4j

* Конфигурация: Файл app.properties (хранение URL, расписания, настроек БД)

## Запуск приложения
1.Убедитесь, что на вашей машине установлены Java 8+ и Maven.

2.Создайте базу данных и настройте подключение к ней в файле app.properties.

3.Укажите периодичность запуска в app.properties (например, cron = 0 0/1 * 1/1 * ? * для запуска раз в минуту).

4.Запустите приложение командой:
mvn clean compile exec:java
